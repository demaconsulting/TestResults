# Introduction

This document records the static analysis results for TestResults.

## Purpose

To provide evidence that TestResults has been analyzed for code quality issues and that any findings have been
reviewed and resolved or accepted.

## Scope

Covers static analysis of all source code in `src/` for TestResults. Test code is excluded from static analysis
requirements. Analysis includes CodeQL security and quality scanning via GitHub Advanced Security and SonarCloud
continuous inspection via SonarScanner for .NET.
